# Study Planner

An AI-powered Study Planner built with React, FastAPI, and PostgreSQL.

## Tech Stack

- React + Vite
- Tailwind CSS
- FastAPI
- PostgreSQL
- SQLAlchemy
- JWT Authentication
- Render
- Vercel

## Live Demo

Frontend:

Backend API:

## Features

- User Registration & Login
- JWT Authentication
- Subject Management
- Task Management
- Weekly Schedule
- Dashboard
